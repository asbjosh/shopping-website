# Opgavebeskrivelse - Udvikling af digital markedsplads for spilnøgler
### Projektoversigt
Formålet med dette projekt er at udvikle en webbaseret shoppingplatform, hvor brugere kan købe digitale spilnøgler fra tredjepartssælgere og aktivere dem på platforme som Steam og Xbox. Systemet skal fungere som en digital markedsplads, hvor sælgere kan oprette og administrere spilnøgler, og købere kan gennemføre sikre køb via en brugervenlig webgrænseflade.

Projektet vil fokusere på full-stack udvikling, API-arkitektur, databasedesign og sikker håndtering af bruger- og transaktionsdata.

## Teknolgistak
**Frontend:** Angular <br />
**Backend:** C# ASP.NET Core Web API <br />
**Database:** PostgreSQL <br />
